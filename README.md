# Run and deploy your AI Studio app (zbudowane z Google AI / Gemini template)

This contains everything you need to run your app locally. Projekt zosta≈Ç wygenerowany z szablonu Google AI Studio (Gemini) i opisuje koncepcyjny frontend TMS/AI dla pracy magisterskiej.

View your app in AI Studio: https://ai.studio/apps/drive/1R1Ds5vAokdiDt-0d4iG6-Xvh2KA9WY_H

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
